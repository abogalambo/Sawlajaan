الحُرُوفُ المركبة
Composite letters		 استخدم - للفصل بين s و h حتَى لَا تترجم إلَى "ش"
Use - to separate between s and h so they don't get converted to "ش" 		s-hr : سهر | shr : شر
وَاو الجَمَاعَة
waaw al-jamaa'a		 اكتُب وَاو الجَمَاعة هكذَا w-a / oo-a لتتجنب إضَافَة الفَتحَة إلَى الوَاو.
Write waaw aljamaa'a like this w-a / oo-a to avoid adding "  َ  " to the waaw. 		kataboo-a : كَتَبُوا ✓
katabooaa : كَتَبُوَا ✖
ب + ال اكتُب ب + ال هكذَا be-al / b-al لتتجنب إضَافَة الفَتحَة إلَى البَاء.
Write ب + ال like this be-al / b-al to avoid adding "  َ  " to the baa. 		be-almnzl : بِالمنزل ✓
baalmnzl : بَالمنزل ✖
